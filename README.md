# Fazzk (치지직 팔로우 알림)

**Fazzk**는 치지직(Chzzk) 스트리머를 위한 **실시간 팔로우 알림 프로그램**입니다.  
방송 화면(OBS)에 투명 배경으로 띄울 수 있으며, 알림 소리와 디자인을 자유롭게 커스터마이징할 수 있습니다.

![Fazzk Icon](public/dodoroi_icon.png)

## ✨ 주요 기능 (Features)

*   **실시간 알림**: 새 팔로워가 생기면 즉시 화면에 알림을 띄웁니다. (재팔로우 감지 포함)
*   **OBS 완벽 지원**:
    *   배경이 투명하게 처리되어 게임 화면 위에 깔끔하게 올라갑니다.
    *   OBS '윈도우 캡처' 또는 '브라우저 소스'로 간편하게 추가 가능합니다.
*   **로그인 유지 (Auto-Login)**:
    *   한 번 로그인하면 프로그램을 껐다 켜도 자동으로 로그인됩니다.
    *   **크롬 확장 프로그램**을 통한 간편 로그인 지원.
    *   **수동 로그인** 지원 (쿠키 직접 입력).
*   **강력한 커스터마이징**:
    *   **사운드**: 원하는 MP3 파일을 알림음으로 설정할 수 있습니다.
    *   **애니메이션**: 페이드(Fade), 슬라이드(Slide), 바운스(Bounce) 효과 지원.
    *   **텍스트**: 글자 색상과 크기를 방송 분위기에 맞게 조절 가능.
*   **안정성**:
    *   **중복 실행 방지**: 이미 앱이 켜져 있다면 새 창을 띄우지 않고 기존 창을 활성화합니다.
    *   **자동 복구**: 네트워크 오류 등으로 연결이 끊겨도 자동으로 재연결을 시도합니다.
*   **연결 상태 표시 (Connection Status)**: OBS 화면 상단에 현재 상태를 직관적으로 표시합니다.
    *   ✅ **연결됨 - 팔로워 알림 대기 중**: 정상적으로 서버와 연결되어 있습니다.
    *   ⚠️ **세션이 만료되었습니다**: 네이버 로그인이 필요합니다.
    *   🔄 **재연결 중...**: 일시적인 연결 끊김으로 자동 복구를 시도하고 있습니다.

## 🚀 설치 및 실행 (Installation)

### 실행 파일 (Windows)
[Releases](https://github.com/minseok7891/fazzk/releases) 페이지에서 최신 버전의 `Fazzk Setup.exe`를 다운로드하여 설치하세요.

### 직접 빌드하기 (Build from source)
Node.js가 설치되어 있어야 합니다.

```bash
# 1. 저장소 복제
git clone https://github.com/minseok7891/fazzk.git
cd fazzk

# 2. 의존성 설치
npm install

# 3. 빌드 (Windows용 exe 생성)
npm run build
```

## 📖 사용 방법 (Usage)

### 1. 로그인 (Login)
프로그램을 실행하면 로그인 화면이 나타납니다. 두 가지 방법 중 하나를 선택하세요.

*   **자동 로그인 (추천)**: '치지직 로그인' 버튼을 눌러 네이버에 로그인합니다.
*   **수동 로그인**:
    1.  함께 제공되는 **Fazzk Extension** (크롬 확장 프로그램)을 브라우저에 설치합니다.
    2.  네이버에 로그인한 상태에서 확장 프로그램 아이콘을 클릭하여 쿠키 정보를 복사합니다.
    3.  Fazzk 앱의 '수동 로그인 (문제 해결용)' 버튼을 누르고 복사한 JSON을 붙여넣습니다.

### 2. 설정 (Settings)
*   우측 하단의 **톱니바퀴(⚙️)** 버튼을 눌러 설정창을 엽니다.
*   알림음, 텍스트 색상/크기, 애니메이션 효과를 설정하고 '알림 테스트'를 해보세요.

### 3. OBS 연동 (OBS Setup)
1.  OBS 소스 목록에서 **[브라우저]** (Browser Source)를 추가합니다.
2.  **URL**: `http://localhost:3000/pages/notifier.html?obs=true`
    *   *참고: 앱 내 '사용 방법' 메뉴에서 현재 실행 중인 정확한 주소를 확인할 수 있습니다.*
3.  **너비(Width)**: `1920` (또는 방송 해상도에 맞게)
4.  **높이(Height)**: `1080`
5.  **커스텀 CSS**: 기본값 유지 (지워도 됨)

## 🛠 기술 스택 (Tech Stack)

*   **Electron**: 데스크탑 앱 프레임워크
*   **Node.js**: 백엔드 로직
*   **Alpine.js**: 가볍고 빠른 프론트엔드 인터랙션
*   **Glassmorphism CSS**: 현대적이고 깔끔한 디자인
