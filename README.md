# Fazzk (치지직 팔로우 알림)

**Fazzk**는 치지직(Chzzk) 스트리머를 위한 **실시간 팔로우 알림 프로그램**입니다.  
방송 화면(OBS)에 투명 배경으로 띄울 수 있으며, 알림 소리와 디자인을 자유롭게 커스터마이징할 수 있습니다.

![Fazzk Icon](public/dodoroi_icon.png)

## ✨ 주요 기능 (Features)

*   **실시간 알림**: 새 팔로워가 생기면 즉시 화면에 알림을 띄웁니다.
*   **OBS 완벽 지원**:
    *   배경이 투명하게 처리되어 게임 화면 위에 깔끔하게 올라갑니다.
    *   OBS '윈도우 캡처'로 간편하게 추가 가능합니다.
*   **로그인 유지 (Auto-Login)**:
    *   한 번 로그인하면 프로그램을 껐다 켜도 자동으로 로그인됩니다.
*   **강력한 커스터마이징**:
    *   **사운드**: 원하는 MP3 파일을 알림음으로 설정할 수 있습니다.
    *   **애니메이션**: 페이드(Fade), 슬라이드(Slide), 바운스(Bounce) 효과 지원.
    *   **텍스트**: 글자 색상과 크기를 방송 분위기에 맞게 조절 가능.
*   **도도로이 에디션**: 귀여운 도도로이 아이콘이 적용되어 있습니다.

## 🚀 설치 및 실행 (Installation)

### 실행 파일 (Windows)
`dist/win-unpacked/Fazzk.exe` 파일을 실행하면 바로 사용할 수 있습니다.

### 직접 빌드하기 (Build from source)
Node.js가 설치되어 있어야 합니다.

```bash
# 1. 저장소 복제
git clone https://github.com/minseok7891/fazzk.git
cd fazzk

# 2. 의존성 설치
npm install

# 3. 빌드 (Windows용 exe 생성)
npm run build
```

## 📖 사용 방법 (Usage)

1.  **로그인**: 프로그램을 처음 실행하면 네이버 로그인 창이 뜹니다. 치지직 계정으로 로그인해주세요.
2.  **설정 (Settings)**:
    *   우측 하단의 **톱니바퀴(⚙️)** 버튼을 눌러 설정창을 엽니다.
    *   알림음, 텍스트 색상/크기, 애니메이션 효과를 설정하고 '알림 테스트'를 해보세요.
3.  **OBS 연동**:
    *   OBS 소스 목록에서 **[브라우저]** (Browser Source)를 추가합니다.
    *   **URL**: `http://localhost:3000/pages/notifier.html?obs=true`
    *   **너비(Width)**: `1920` (또는 방송 해상도에 맞게)
    *   **높이(Height)**: `1080`
    *   **커스텀 CSS**: 기본값 유지 (지워도 됨)
    *   *참고: 포트(3000)는 프로그램 실행 시 바뀔 수 있으니, 만약 안 뜨면 앱 설정에서 포트를 확인하세요.*
