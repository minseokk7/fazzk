### 새로운 기능
-  자동 업데이트: 앱 시작 시 GitHub Releases에서 새 버전 확인
-  세션 유지 개선: 10분마다 세션 자동 저장
-  만료 임박 경고: 쿠키 만료 1시간 전 알림
-  자동 재연결: 연결 오류 시 최대 5회 자동 재시도

### 트레이 메뉴
- '업데이트 확인' 항목 추가

### 확장프로그램
-  연결 상태 아이콘: 연결(초록), 미연결(회색), 동기화중(주황)
-  30초마다 연결 상태 자동 확인

### 확장 프로그램 상태 표시 (Extension Status)
확장 프로그램 아이콘 위 **뱃지(Badge)**의 색상과 문자로 상태를 확인할 수 있습니다.
* 🟩 **(초록색 배경)**: **연결됨** - Fazzk 앱과 정상적으로 통신 중입니다.
* ❗ **! (빨간색 배경)**: **미연결** - 앱이 실행되지 않았거나 포트를 찾을 수 없습니다.
* 🔄 **↻ (주황색 배경)**: **동기화 중** - 네이버 로그인 정보를 앱으로 전송하고 있습니다.
