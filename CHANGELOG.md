## [2.1.0] - 2025-12-15

### ✨ 주요 변경사항
- **⚡ 성능 최적화 (Rust)**:
  - HTTP 클라이언트 재사용 로직 구현 (메모리/네트워크 효율 증가)
  - 불필요한 벡터 할당 및 이터레이터 연산 최적화 (`clippy` 적용)
- **🎨 UX 개선**:
  - `alert()` 팝업을 **네이티브 스타일 Toast 알림**으로 전면 교체
  - 다크/라이트 모드에 대응하는 깔끔한 애니메이션 적용
- **🛠 코드 리팩토링**:
  - JS API 네이밍 변경: `window.electronAPI` → `window.fazzkAPI` (Electron 의존성 완전 제거)
  - 레거시 코드 및 미사용 파일 정리
- **🐛 버그 수정**:
  - 트래킹 방지 기능으로 인한 **폰트(Pretendard) 로딩 실패** 문제 해결 (로컬 폰트 내장)
  - CI/CD 빌드 스크립트 오류 수정 (`require` ESM 호환성)

## [2.0.0] - 2025-12-15

  - 알림 소리 사용자 커스터마이징
  - 시스템 테마 자동 감지
- **버그 수정**:
  - Windows 앱 아이콘 적용 문제 해결 (리소스 경로 최적화)
